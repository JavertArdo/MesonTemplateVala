# MesonTemplateVala
Simple Meson Template Project
* Vala
* Meson

## Dependencies
### Elementary OS
* valac
* meson
* build-essential
* libgtk-3-dev
* libgee-0.8-dev
* libsoup2.4-dev

## Build
Type the following commands manually
```
> meson build
> cd build
> ninja
```
or run script which rebuilds whole project and runs application
```
> chmod +x run.sh
> ./run.sh
```
